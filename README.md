# Ejecutar en Desarrollo
1. Clonar el repositorio
2. Ejecutar `npm install`
3. Ejecutar `npm run start:dev`
4. Tener NestJS CLI instalado `npm i -g @nestjs/cli`
5. Levantar la base de datos con `docker-compose up -d`

## Stack Usado
- NestJS
- TypeORM
- MongoDB
- Docker
